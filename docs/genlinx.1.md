# NAME

genlinx - an open source CLI tool for NetLinx projects

# SYNOPSIS

**genlinx** [*CMD*] [*OPTIONS*]\
**genlinx archive** [*OPTIONS*]\
**genlinx build** [*OPTIONS*]\
**genlinx cfg** [*OPTIONS*]\

# DESCRIPTION

**genlinx** is a CLI helper tool for work with NetLinx Studio projects

# EXAMPLES

**genlinx --help**
: Display help information

# EXIT VALUES

**0**
: Success

**!0**
: Failure
